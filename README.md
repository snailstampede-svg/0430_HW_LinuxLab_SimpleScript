# 0430 HW Linux Lab Simple Script Assignment

## Objective

Create a basic executable shell script with at least 12 commands issued

## Basic Script Functionality  

This script was designed to back up a folder and any files contained therein. 

The script will perform the following functions:

1. Create a directory and add files inside that directory.
2. Inform the user of the steps the script is taking.
3. Create a directory in which to save backup versions of those files.
4. Create a separate directory and backup of those files in that new directory
5. Display the contents of the backup files to the user.
6. Display the character count and save that character count to a separate file.

## How to use this script

1. Copy this script to a folder
2. Ensure that you are allowed to execute this script by running the following command.

    ``` bash
    chmod +x simplescript.sh
    ```

3. At the command prompt, type the following:  

    ```bash
    ./simplescript.sh
    ```

>[! Note]
> The script will execute and display its steps to the screen. You can check the results in the created subfolders within your current folder.
